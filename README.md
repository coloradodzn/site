# Progetto D'esame Sito

## Obiettivo

Un sito web personale statico che rappresenti un portfolio professionale completo, pronto per essere pubblicato online. Il sito è composto da più pagine HTML/CSS, deve essere graficamente coerente, responsive, semanticamente corretto e ottimizzato per SEO.

## Stack Tecnologico

- **HTML5** — struttura semantica delle pagine
- **CSS3** — stilizzazione, layout responsive, animazioni
- **JavaScript vanilla** — interattività e comportamenti dinamici
- **Google Fonts** — tipografia web

## Struttura delle Pagine

| File | Descrizione |
|------|-------------|
| `index.html` | Homepage: presentazione principale e introduzione al portfolio |
| `bio.html` | Pagina biografia: informazioni personali, background e competenze |
| `portfolio.html` | Pagina portfolio: griglia dei lavori realizzati con anteprima |
| `lavoro-1.html` | Pagina di dettaglio del primo progetto del portfolio |
| `lavoro-2.html` | Pagina di dettaglio del secondo progetto del portfolio |
| `lavoro-3.html` | Pagina di dettaglio del terzo progetto del portfolio |
| `contatti.html` | Pagina contatti: form di contatto e link ai profili social |

## Struttura delle Cartelle

```
/css
```
Contiene i fogli di stile del progetto. Il file principale `style.css` gestisce il reset, le variabili CSS, il layout globale, la tipografia, i componenti riutilizzabili e le media query per la responsività.

```
/img
```
Contiene tutte le risorse grafiche del sito: foto del profilo, immagini di copertina dei progetti, icone e qualsiasi altro asset visivo utilizzato nelle pagine.

```
/js
```
Contiene i file JavaScript del progetto. Il file principale `main.js` gestisce l'interattività del sito: navigazione mobile, animazioni scroll, comportamenti UI e qualsiasi altra logica lato client.
